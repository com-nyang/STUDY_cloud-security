# 방화벽 > 인터넷 접속 제한 서비스

## 위치

`방화벽 > 인터넷 접속 제한 서비스`

## 기능 설명

인터넷 접속 제한 서비스 설정을 할 수 있는 페이지입니다.

## 보통 여기서 보는 항목

- 인터넷 접속 제한 서비스
- 인터넷 접속 제한 목록

|인터넷 접속 제한 목록|

| No  | 사용  | 단말이름 | 단말주소(MAC) | 요일(시간) | 허용/차단 | 삭제  |
| --- | --- | ---- | --------- | ------ | ----- | --- |
|     |     |      |           |        |       |     |

- 허용 : 설정된 시간만 인터넷 이용이 가능하며 나머지 시간은 인터넷 이용 차단

- 차단 : 설정된 시간만 인터넷 이용이 차단되며 나머지 시간은 인터넷 이용 가능 동일 단말에 대해서는 한가지 규칙만 설정 가능

- 적용 : 리스트에서 "사용/삭제" 항목을 선택하시고, "적용"하셔서 사용하실 수 있습니다.

인터넷 접속 제한 서비스 시간에는 인터넷을 사용할 수 없으며, LTE/3G로 전환 시 데이터가 소진될 수 있습니다.
