## 궁금증 설명

### Band

`Band`는 무선 네트워크가 사용하는 주파수 대역을 뜻한다. 보통 `2.4GHz`, `5GHz`처럼 표시된다.

- `2.4GHz`
  도달 거리가 길고 벽 통과가 비교적 잘 되지만, 주변 기기와 간섭이 많을 수 있다.
- `5GHz`
  속도가 빠르고 간섭이 상대적으로 적을 수 있지만, 도달 거리는 더 짧은 편이다.

즉 상태정보에서 `Band`를 보는 이유는 "이 Wi-Fi가 어느 주파수 대역에서 동작 중인가"를 확인하기 위해서다.

### SSID / BSSID

`SSID`는 사람이 보는 Wi-Fi 이름이다. 휴대폰이나 노트북에서 검색할 때 보이는 네트워크 이름이 SSID다.

`BSSID`는 그 Wi-Fi 장치 자체의 고유 식별자에 가까운 값으로, 보통 무선 인터페이스의 MAC 주소 형태로 표시된다.

- `SSID`
  예: `MyHomeWiFi`
- `BSSID`
  예: `AA:BB:CC:11:22:33`

같은 이름의 Wi-Fi가 여러 개 있어도 BSSID는 각각 다를 수 있다. 그래서 SSID는 "이름", BSSID는 "실제 장비 식별값"으로 이해하면 된다.

### 간섭 통계정보 (CU/FA)

이 값은 주변 무선 환경이 얼마나 혼잡한지 보여주는 통계일 가능성이 높다.

- `CU`
  보통 `Channel Utilization`의 약자로 해석한다. 현재 채널이 얼마나 사용 중인지, 즉 얼마나 바쁜지를 뜻한다.
- `FA`
  보통 `False Alarm` 또는 유사한 간섭/잡음 관련 통계로 해석한다. 실제 데이터가 아닌 잡음이나 간섭성 신호를 많이 감지하는 상태를 의미할 수 있다.

쉽게 말하면 이 항목은 "지금 이 채널이 얼마나 붐비는가", "주변 전파 간섭이 어느 정도인가"를 보여주는 참고 정보다.

### DAD 상태

`DAD`는 `Duplicate Address Detection`의 약자다. IPv6에서 같은 주소를 다른 장치가 이미 사용 중인지 확인하는 절차다.

즉, 공유기나 장치가 IPv6 주소를 사용할 때:

1. 이 주소를 쓰려고 한다.
2. 이미 누가 쓰고 있는지 확인한다.
3. 중복이 없으면 사용한다.

따라서 `DAD 상태`는 "이 IPv6 주소가 중복 없이 정상적으로 사용 가능한 상태인가"를 보여주는 값으로 이해하면 된다.

### IGMP

`IGMP`는 `Internet Group Management Protocol`의 약자다. 여러 장치가 같은 방송형 데이터 스트림을 함께 받을 때 사용하는 프로토콜이다.

대표적으로 IPTV, 멀티캐스트 스트리밍 같은 환경에서 많이 등장한다.

- 일반 인터넷 접속은 보통 1:1 통신에 가깝다.
- IGMP는 여러 장치가 같은 스트림을 받아야 할 때 어떤 장치가 그 그룹에 참여하는지 관리한다.

상태정보에 `IGMP`, `Fast Leave`, `Member Expire Time`, `Query Interval`이 보인다면, 이는 멀티캐스트 트래픽을 어떻게 관리하는지 보여주는 항목으로 이해하면 된다.

## 한 번에 이해하기

이 화면은 단순히 "공유기 상태를 보여준다"에서 끝나는 게 아니다. 실제로는 아래처럼 읽을 수 있다.

- `시스템`
  공유기 자체가 얼마나 오래 켜져 있었고 어떤 소프트웨어 버전을 쓰는지 보여준다.
- `Wi-Fi 정보`
  현재 무선 네트워크가 어떤 이름, 어떤 주파수, 어떤 채널에서 동작하는지 보여준다.
- `랜 정보`
  내부 네트워크 주소 체계를 보여준다.
- `인터넷 정보`
  외부 인터넷 연결 정보를 보여준다.
- `IPv6`
  IPv6 연결 상태를 보여준다.
- `IGMP`
  IPTV나 멀티캐스트 같은 특수 트래픽 관련 동작을 보여준다.

즉 `홈 > 상태정보`는 공유기의 현재 운영 상태를 여러 층위에서 요약해서 보여주는 대시보드라고 보면 된다.
