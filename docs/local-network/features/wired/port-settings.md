# 유선설정 > 포트 설정

## 위치

`유선설정 > 포트 설정`

## 기능 설명

각 포트의 속도, 미디어 속성, 네트워크 모드 및 활성화/비활성화 설정을 할 수 있는 페이지입니다.

## 궁금증 설명

### 이 포트는 어느 포트를 말하는건가? 8000과 같은 포트가 아닌지?

여기서 말하는 `포트`는 보통 `TCP/UDP 포트 번호`가 아니라 공유기 뒤에 실제로 꽂는 `물리 포트`를 뜻한다.

즉:

- `WAN`
  인터넷 선이 들어오는 물리 포트
- `LAN1`, `LAN2`, `LAN3`, `LAN4`
  PC나 다른 장비를 연결하는 물리 포트

반면 `8000`, `80`, `443` 같은 것은 `논리 포트 번호`다. 이건 네트워크 프로그램이나 서비스가 통신할 때 사용하는 번호다.

둘은 이름은 같지만 의미가 다르다.

- `물리 포트`
  케이블을 꽂는 실제 단자
- `논리 포트`
  네트워크 서비스가 사용하는 번호

따라서 이 메뉴는 `포트 포워딩` 메뉴와 달리, 서비스 번호를 다루는 곳이 아니라 `공유기 하드웨어 포트 상태`를 다루는 메뉴라고 이해하면 된다.

### LAN 2P500?

이 값은 메뉴 이름이나 모델별 기능 이름일 가능성이 높고, 일반적인 네트워크 표준 용어로 바로 읽히는 항목은 아니다.

가능성은 몇 가지가 있다.

1. 특정 LAN 포트에 연결된 장치 프로필명일 수 있다.
2. 제조사 내부 명칭이나 칩셋 관련 옵션일 수 있다.
3. 포트별 특수 기능이나 연결 상태 표시일 수 있다.

지금 정보만으로는 정확히 단정하기 어렵다. 이 경우에는 아래처럼 확인하는 게 맞다.

- 해당 칸이 `체크박스`인지 단순 텍스트인지 본다.
- 다른 LAN 포트에도 비슷한 문자열이 있는지 본다.
- 마우스를 올렸을 때 설명이 나오는지 본다.
- 값을 체크하거나 해제할 수 있는지 본다.

즉 `LAN 2P500`은 일반 개념 설명보다 `해당 공유기 제조사 UI에서 따로 붙인 항목`일 가능성이 높다.

## 한 번에 이해하기

`유선설정 > 포트 설정`은 서비스 포트 번호를 다루는 메뉴가 아니라, 공유기의 실제 WAN/LAN 단자가 어떤 상태로 동작하는지 보는 메뉴다.

- `협상`
  속도와 전송 방식을 자동으로 맞출지 여부
- `속도`
  100Mbps, 1000Mbps 같은 링크 속도
- `전송방식`
  full / half duplex 같은 통신 방식
- `흐름제어`
  데이터가 몰릴 때 전송을 조절하는 기능
- `전원`
  해당 물리 포트의 활성화 여부
- `리셋`
  그 포트 상태를 다시 초기화하거나 재협상하는 기능일 수 있음

즉 이 메뉴는 "공유기 케이블 포트들이 현재 어떻게 연결되고 동작 중인가"를 보는 페이지라고 이해하면 된다.
