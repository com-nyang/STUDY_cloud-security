# Router Admin Change Log

## Purpose

공유기 관리자 페이지에서 실제로 변경한 설정과 결과를 기록한다. 설정 변경은 반드시 복구 가능성을 같이 남긴다.

## Record Template

### Change Item

- 날짜:
- 메뉴 위치:
- 변경 전 값:
- 변경 후 값:
- 변경 목적:
- 예상 효과:
- 실제 결과:
- 부작용:
- 복구 방법:

## Example Items To Track

- 관리자 비밀번호 변경
- Wi-Fi SSID / 비밀번호 변경
- WPA2/WPA3 설정 변경
- DHCP reservation 추가
- Port forwarding 규칙 추가
- UPnP 비활성화
- 원격 관리 비활성화
- 펌웨어 업데이트

## Safety Rules

- 한 번에 하나씩만 변경한다.
- 변경 전 현재 값을 먼저 기록한다.
- 외부 접속 관련 설정은 테스트 방법을 같이 기록한다.
- 관리자 접근이 끊길 수 있는 설정은 복구 경로를 먼저 확인한다.
